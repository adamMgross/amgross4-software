Windows Requirements:

  • Csound 5.10+
  • Max 5+
  • Windows XP/Vista

    Mac Requirements:

  • Csound 5.10+ or later
  • Max 5+
  • MacOS 10.4+ (intel)

    2010_09_01
    Version 1.1.0 released. View the change log.

    2009_10_12
    csound~ 1.0.7 released.

    2009_07_21:
    csound~ 1.0.6 released. Some new features have been added and the .help and .maxhelp patches have been improved. [dp.pathconvert] is no longer included in the zip file as it's no longer necessary on MacOSX.

    2009_01_21:
    csound~ 1.0.5 released. This version REQUIRES Csound 5.10 or later.
    Doubles version now available for intel Mac.

    2008_10_19:
    Max5 versions of all help and example patches added to v1.0.4 package.
    Csound 5.09.2 for Windows has been released. It fixes "WARNING: could not open library" and "Error 998" errors.

    2008_06_24: v1.0.4 (build 1) released.

    2007_06_20: csound~_v1.0.3.zip has been released.

    2007_05_23: csound~_v1.0.2.zip has been released.

    2007_05_10: csound~_v1.0.1.zip has been updated with new binaries that should fix the "error 14001" problem on WinXP and the "csound~ doesn't load and doesn't report an error" problem on Mac (ppc) + MaxMSP4.5.7.

    2007_04_01: An Intel Mac build of csound~ has been added to csound~_1.0.1.zip. Go here to download Csound5 for Intel Mac.

    2007_03_09: csound~ versions will use a new numbering system. The choice of 1.0 is arbitrary; the change is being made for simplicity sake.

    The zip file contains the binaries WinXP and MacOSX (intel/ppc), source code, install instructions, and a demo patch with tutorial. The demo patch (shown below) does not demonstrate every feature of csound~. See the manual for detailed information.

    csound~ now supports most features found in Ingalls' csound~. Although there are differences, most users who upgrade should not have a problem. The most notable difference is the outlet configuration; see manual for details.
