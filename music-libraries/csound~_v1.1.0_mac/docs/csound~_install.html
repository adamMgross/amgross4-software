Installing csound~

These are the instructions for installing csound~ v1.1.0.
Instructions for earlier versions are contained within their respective archive files.

Step 1: Install Csound5

csound~ depends on Csound5.

You can download Csound5 from here.

I recommend downloading an automatic installer (ends with .exe or .dmg). Take note of the version (floats or doubles). If the download file contains "double" or "d", it's doubles. If the download file contains "float" or "f", it's floats. If it contains neither, then it probably contains both, in which case you don't have to worry it.

Step 2: Download and run the csound~ installer.

On Windows, download csound~_v1.1.0_win_double.zip or csound~_v1.1.0_win_float.zip and expand it. Then run the exe.

On MacOSX, download csound~_v1.1.0_mac.zip and expand it. Then choose either csound~_v1.1.0_double.pkg or csound~_v1.1.0_float.pkg and run it.

You are now done installing csound~. However, if you don't want to use the automatic installer, continue to Step 3.

If you skipped Step 2, then...

Step 3: Download csound~_v1.1.0.zip file and manually install.


Mac OS X (intel only)

1 Expand the zip file.
2 Navigate to the mac folder.

Copy csound~_double.mxo or csound~_float.mxo (depending on which version of Csound5 you installed) to:

/Applications/Max5/Cycling '74/msp-externals/

Rename the file csound~.mxo.

3 Navigate to help_files folder.

Copy all .csd files, csound~.maxhelp to the help folder.  

For MaxMSP 5.0 (and later), the help folder is:

/Applications/Max5/Cycling '74/msp-help/
If you have previously installed csound~.mxo, make sure that only one copy of csound~.mxo exists in Max's search path. Otherwise, you may accidently load an older version.

If you have prevously installed Matt Ingalls' csound~, be sure to remove it from the externals folder.

To check your installation, open the patch "csound~.maxhelp". In the Max window, you should see the csound~ version. It will look something like this:

     csound~ v1.1.0

If you don't see it, then you have not installed csound~.mxo properly, or Matt Ingalls' csound~ is being loaded instead of csound~.mxo.


Windows XP/Vista/7

1 Expand the csound~ zip file.
2 Navigate to the win sub-directory.

a) Copy either csound~_float.mxe or csound~_double.mxe (depending on your Csound5 version) to your MaxMSP externals folder.

For MaxMSP 5.0 (and later), the externals folder is:

C:\Program Files\Cycling '74\Max 5.0\Cycling '74\msp-externals\
b) Rename the file csound~.mxe.
3 Navigate to help_files folder and copy all .csd files and csound~.maxhelp to the msp-help folder.

For MaxMSP 5.0 (and later), the help folder is:

C:\Program Files\Cycling '74\Max 5.0\Cycling '74\msp-help\

If you have previously installed csound~.mxe, make sure that only one copy of csound~.mxe exists in Max's search path. Otherwise, you may accidently load an older version.

To check your installation, open the patch csound~.maxhelp. In the Max window, you should see the csound~ version. It should look something like this:

     csound~ v1.1.0

If you don't see it, then you have not installed csound~.mxe properly.
